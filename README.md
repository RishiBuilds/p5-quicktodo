# P5 – QuickTodo

A responsive Todo List application built using HTML, CSS, and JavaScript that helps users manage daily tasks efficiently.

## Features
- Add, edit, and delete tasks
- Mark tasks as completed
- Filter tasks (All / Active / Completed)
- Task statistics (Total, Active, Completed)
- Persistent storage using localStorage
- Light & dark theme toggle
- Keyboard support (Enter to add/edit, Escape to cancel)

## Tech Stack
- **HTML5**, **CSS3**, **JavaScript (Vanilla)** 

## How It Works
1. User adds a task 
2. Tasks are stored locally in the browser
3. Users can complete, edit, filter, or delete tasks
